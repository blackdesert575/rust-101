# HackerRank in Rust

This directory contains my Rust solutions to HackerRank problems, organized by domain and difficulty.

## 📁 Structure

- `Problem Solving/`
  - `Basic/`: Beginner-level problems
  - `Intermediate/`: Mid-level challenges
  - `Advanced/`: Complex or algorithmic puzzles

## 🛠 Notes

Each problem is implemented as a standalone Rust binary using Cargo.  
Solutions include:
- Clean and idiomatic Rust code
- Comments explaining the approach
- (Optional) Basic test cases if applicable

## 🧩 How to Run

Navigate into a problem directory and run:

```bash
cargo run
```
